# Life Simulator

A complex biological and social simulation running in the browser.

**Live Demo:** [https://sim.ewrdad.uk/](https://sim.ewrdad.uk/)

## Features
- **Ecosystem Simulation:** Wolves, cows, and plants interacting in a dynamic environment.
- **Human Civilization:** Humans that build, gather resources, and form factions.
- **Genetics System:** Traits passed down through generations affecting survival and behavior.
- **Random Events:** Dynamic events that shake up the world state.

## Development

This project is built with React and Vite.

### Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

### Testing
Run the test suite with: `npm test`
